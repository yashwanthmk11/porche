# Porsche Website Clone

A responsive **Porsche website clone** built using **HTML, CSS, and JavaScript** that replicates the premium design and layout of the official Porsche website.  
The project demonstrates frontend development skills and basic cloud deployment.

## 🚀 Technologies Used
- HTML5
- CSS3
- JavaScript
- AWS EC2
- Apache Tomcat

## ☁️ Deployment
The project is deployed on an **AWS EC2 instance using Apache Tomcat server**.
